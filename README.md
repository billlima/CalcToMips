# CalcToMips

This software uses RPN (Reverse Polish Notation) to convert simple numerical calculations to a Mips Code.

&nbsp;

**Formatos aceitos:**
* sum(+)
* sub(-)
* mult(*)
* div(/)
* fat(n)
* sqrt(n)
* fib(n)

**Examples**
```
50 + 13 / (25 - 5) - fib(6)
(25 * (13 + 2)) - fat(5)
922 - fat(6) * (3 - 1) + fib(12) + 75 / sqrt(25)
```
&nbsp;
&nbsp;

* Tomcat7+
